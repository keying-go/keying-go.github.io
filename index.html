CV | Facial Expression Detection

We implemented 4 models in total to classify the input facial expression images into one of the 7 categories (angry, disgust, fear, happy, neutral, sad, and surprise). We first implemented a self-designed Convolutional Neural Network (CNN) that achieved 63.19% accuracy. Then, to further improve the accuracy of expression detection, we leveraged transfer learning by having 3 backbone structures (VGG-16, MobileNet, and ResNet50) along with our own classification heads. The optimal structure achieved a test accuracy of 69.47%. Our system should be used with individuals' explicit consent. It should not be misused for malicious activities.
